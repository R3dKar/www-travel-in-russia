# Путешествия по России

Лендинг о крассивых местах в России, куда можно съездить в отпуск.

### Использованные технологии:
 - анимации (transition)
 - flexbox
 - grid
 - `normalize.css`

# Варианты запуска
 - [GitHub Pages](https://r3dkar.github.io/www-travel-in-russia/)
 - Скачать проект, открыть `index.html`
